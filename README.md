# Paul Cahen — personal website

Source code for my personal academic website:

https://paul-cahen.github.io/

This site is built with the Academic Pages Jekyll template.

## Main files

- `_config.yml`: site metadata and sidebar information.
- `_pages/about.md`: homepage.
- `_pages/publications.md`: publications and preprints.
- `_pages/talks.md`: talks.
- `_pages/cv.md`: CV page.
- `files/`: PDFs.
- `images/profile.jpg`: profile picture.
